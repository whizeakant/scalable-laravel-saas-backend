# Scalable SaaS Backend

## 🚀 SaaS Billing System

Implemented a production-style billing architecture including:

* Subscription lifecycle management
* Stripe webhook processing
* Invoice persistence system
* Event-driven architecture
* Idempotent webhook handling
* Automatic retry & background jobs

### 🏗️ Architecture Highlights

* Event store for webhook reliability
* Fault-tolerant payment processing
* Background retry workers using cron jobs
* Modular service-based backend design

Built to simulate real-world SaaS platforms like Stripe-integrated products.
